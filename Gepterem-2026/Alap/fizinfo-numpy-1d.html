Pár gondolat a NumPy-ről¶

NumPy: Numerical Python modul. Szokásos hívása: import numpy as np.

  • https://numpy.org/
  • https://en.wikipedia.org/wiki/NumPy

Nagyon sokat tudó modul, mely lehetőséget ad Pythonban (relatíve) gyors numerikus számításokra. Megoldás arra, hogy a Python listák (rugalmasságuk miatt) nem hatékonyak, ha sok, azonos típusú értéket tárolunk bennük és ezekkel végzünk műveletet.

Számumnkra fontos NumPy alap típusok:

  • np.float32, np.float64: 32 illetve 64 bites lebegőpontos szám
  • np.int32, np.int64: 32 illetve 64 bites egész
  • np.bool_: logikai érték (True vagy False)

A NumPy alap adat-struktúrája a tömb (array), amik csupa azonos típusú elemből állnak. Főbb sajátosságok:

  • a listákhoz hasonlóan indexelhetők
  • támogatják az úgynevezett "vektorizált" végrehajtást, azaz amikor operátor vagy függvény automatikusan végrehajtódik minden elemre; pl. ha arr1 és arr2 két, azonos méretű NumPy tömb, akkor arr1-arr2 egy ugyanekkora tömb, elemenként kiszámolva a különbséget
  • sok művelet egy paranccsal elérhető, pl. arr.mean() a tömbelemek átlaga, arr.std() a szórás (standard deviation)
  • sok algoritmus, pl. regresszió, interpoláció, egyenlet-megoldás, be van programozva

A használat többnyire elég kézenfekvő. Érdemes megtanulni, mert sokkal tömörebb és gyorsabb a kód, mintha Python list-ekkel és ciklusokkal dolgoznánk. (De egy C++-ban írt és lefordított program sebességét nem éri el.)

Sok más projekt épít erre, pl. SciKit-Learn, SciPy (Scientific Python), OpenCV (Open Computer Vision), Pandas, ...

Az új eljárások gyűjteménye¶

Tömb létrehozás:

  • aa=np.array(lista, dtype=típus): adott lista elemeit a típus-ra alakítja és ebből hoz létre tömböt
  • aa=np.zeros(N, dtype=típus): N méretű, csupa 0-ból álló tömb
  • bb=np.zeros_like(aa): ugyanakkora és ugyanolyan típusú tömb létrehozása, mint aa
  • dd=np.append(aa, bb): két tömb egymás után fűzése és tárolása
  • cc=np.arange(start, end, delta, típus): start és end közt delta egyenletes lépésekkel létrejövő tömb
  • cc2=np.linspace(start, end, N, típus): start és end közt egyenletes lépésekkel létrejövő N elemű tömb

Matemetikai függvények:

  • Műveleti jelek: +, -, *, /, **, ...: NumPy tömbökre elemenként vonatkoznak
  • np.radians(): fokról radiánra vált
  • np.sin(), np.cos(), np.abs(), ...: szinusz, koszinusz, abszolút érték, ...
  • vfun=np.vectorize(fun): előállítja a fun vektorizált változatát

Összegzés:

  • aa.sum(): a tömb összes elemének összege
  • aa.diff(): szomszédok különbségéből álló tömb
  • aa.cumsum(): egy tömb fokozatos felösszegzése (kumulatív szumma)

Keresés:

  • aa.max(), aa.min(): a tömb maximális/minimális elemének nagysága
  • aa.argmax(), aa.argmin(): a tömb maximális/minimális elemének indexe (hányadik helyen van)
